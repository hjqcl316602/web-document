# pay

## 支付宝

### 截频扫码转账（同微信方式）

- 该方式同微信一致
- 该方式不需要显示真实姓名

### 唤起 app 转账，并设置金额方式

- 该方式有一定概率的手机会出现不能跳转成功（解决办法：增加一个扫一扫功能，即让用户转账之前使用一下外置摄像头功能）

### 注意

- 若支付宝二维码截屏，有的手机则不能直接进入支付进行扫码支付，会出现由于系统原因，需要单独进入支付宝选择才行
